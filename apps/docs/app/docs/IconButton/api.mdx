# API

The IconButton component provides compact, accessible action triggers for icon-only interactions. It extends Button with specialized behavior for visual symbols, enforcing accessibility requirements, providing automatic square sizing, and including built-in tooltip integration.

## Import

```tsx
import { IconButton } from '@kushagradhawan/kookie-ui';
```

## Props

IconButton inherits all Button props but adds specialized accessibility requirements for icon-only interactions.

## Icons

IconButton automatically sizes icons based on the button size. Never manually size icons in IconButton.

<CodeBlock
  preview={
    <Flex gap="2" align="center">
      <IconButton size="1" variant="soft" aria-label="Settings">
        <Settings />
      </IconButton>
      <IconButton size="2" variant="soft" aria-label="Settings">
        <Settings />
      </IconButton>
      <IconButton size="3" variant="soft" aria-label="Settings">
        <Settings />
      </IconButton>
      <IconButton size="4" variant="soft" aria-label="Settings">
        <Settings />
      </IconButton>
    </Flex>
  }
>

```tsx
{/* Icons are automatically sized */}
<IconButton size="1" variant="soft" aria-label="Settings">
  <Settings />
</IconButton>

{/* Don't manually size icons */}
<IconButton size="2" variant="soft" aria-label="Settings">
  <Settings size={16} /> {/* ❌ Don't do this */}
</IconButton>
```

</CodeBlock>
