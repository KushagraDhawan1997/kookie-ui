# Fonts

Customize fonts throughout your application using theme props and CSS variables. Switch between sans-serif and monospace fonts easily, or replace fonts entirely with your preferred choices.

<CodeBlock buttonsPosition="center" >

```tsx
import { Theme } from '@kushagradhawan/kookie-ui';
```
</CodeBlock>

</Section>

<Section size="1">

## Theme

Use the `fontFamily` prop to switch between sans-serif and monospace fonts:

<CodeBlock>

```tsx
// Sans-serif fonts (default)
<Theme fontFamily="sans">
  <Text>This uses Inter font</Text>
</Theme>

// Monospace fonts
<Theme fontFamily="mono">
  <Text>This uses JetBrains Mono font</Text>
</Theme>
```
</CodeBlock>

</Section>

<Section size="1">

## Override CSS

Override CSS variables to use your preferred fonts:

<CodeBlock>

```css
/* In your global CSS file */
:root {
  /* Customize sans-serif fonts */
  --font-sans: 'Poppins', 'Inter', -apple-system, BlinkMacSystemFont, sans-serif;
  
  /* Customize monospace fonts */
  --font-mono: 'Fira Code', 'JetBrains Mono', 'Consolas', monospace;
}
```
</CodeBlock>

</Section>

<Section size="1">

## Font Replacement

<CodeBlock>

```css
/* Import your preferred fonts */
@import url('https://fonts.googleapis.com/css2?family=Poppins:wght@300;400;500;600;700&display=swap');
@import url('https://fonts.googleapis.com/css2?family=Fira+Code:wght@300;400;500;600;700&display=swap');

/* Override Kookie UI font variables */
:root {
  /* Replace base font definitions */
  --font-inter: 'Poppins', -apple-system, BlinkMacSystemFont, sans-serif;
  --font-jetbrains-mono: 'Fira Code', 'JetBrains Mono', monospace;
  
  /* Update theme font mappings */
  --font-sans: var(--font-inter);
  --font-mono: var(--font-jetbrains-mono);
}
```
</CodeBlock>

## Programmatic Configuration

For dynamic font switching, use the provided utilities:

<CodeBlock>

```tsx
import { applyFontConfig, FONT_STACKS } from '@kushagradhawan/kookie-ui';

// Apply Google Fonts
applyFontConfig({
  sans: FONT_STACKS.poppins,
  mono: FONT_STACKS.firaCode,
});

// Use system fonts for performance
applyFontConfig({
  sans: FONT_STACKS.system,
  mono: FONT_STACKS.systemMono,
});
```
</CodeBlock>

<Text size="2" color="gray">
Available font stacks: <Code>inter</Code>, <Code>system</Code>, <Code>poppins</Code>, <Code>openSans</Code>, <Code>jetbrainsMono</Code>, <Code>firaCode</Code>, <Code>sourceCodePro</Code>, <Code>systemMono</Code>.
</Text>
