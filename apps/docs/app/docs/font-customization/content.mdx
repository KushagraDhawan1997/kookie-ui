# Font Customization

Kookie UI makes font customization simple and flexible. Whether you're designing a brand-specific interface or need to match existing design systems, you can easily customize fonts without complex configuration.

</Section>

<Separator size="4"/>

<Section size="1">

## Why It Matters

Typography is fundamental to interface design. The right font choice can:

- **Establish brand identity** through consistent typography across your application
- **Improve readability** by choosing fonts optimized for your content and audience
- **Create visual hierarchy** using different font weights and styles
- **Enhance accessibility** with fonts that work well across different devices and screen sizes

Kookie UI's font system is designed to be flexible enough for any design requirement while maintaining consistency across all components.

</Section>

<Separator size="4"/>

<Section size="1">

## Built-in Fonts

Kookie UI comes with carefully selected default fonts that work well together:

### Sans-serif Fonts
- **Inter** - Clean, modern, and highly readable for UI text
- **System fonts** - Native fonts that feel familiar to users on each platform

### Monospace Fonts  
- **JetBrains Mono** - Designed specifically for code and technical content
- **System monospace** - Native monospace fonts for optimal performance

These fonts are chosen for their excellent readability, wide character support, and professional appearance across different contexts.

</Section>

<Separator size="4"/>

<Section size="1">

## Font Switching

You can switch between font families instantly using simple theme props. This makes it easy to:

- **Test different fonts** during the design process
- **Create font variations** for different sections of your application
- **A/B test typography** to see what works best for your users

<CodeBlock>

```tsx
// Sans-serif fonts (default)
<Theme fontFamily="sans">
  <Text>This uses Inter font</Text>
</Theme>

// Monospace fonts
<Theme fontFamily="mono">
  <Text>This uses JetBrains Mono font</Text>
</Theme>
```
</CodeBlock>

## Custom Fonts

Want to use your brand fonts? Kookie UI makes it straightforward to integrate any font family:

### What You Can Customize
- **Primary text fonts** for headings, body text, and UI elements
- **Code fonts** for technical content and data display
- **Font weights** and styles to match your design system
- **Fallback fonts** to ensure text displays correctly across all devices

### How Easy It Is
Font customization requires minimal technical knowledge. You simply:
1. Import your preferred fonts (Google Fonts, custom fonts, etc.)
2. Update a few CSS variables
3. Your entire application uses the new fonts automatically

This means designers can specify any font they want without worrying about complex implementation details.

</Section>

## Implementation

For developers implementing font changes, here are the technical details:

### CSS Variable Override

<CodeBlock>

```css
/* In your global CSS file */
:root {
  /* Customize sans-serif fonts */
  --font-sans: 'Poppins', 'Inter', -apple-system, BlinkMacSystemFont, sans-serif;
  
  /* Customize monospace fonts */
  --font-mono: 'Fira Code', 'JetBrains Mono', 'Consolas', monospace;
}
```
</CodeBlock>

### Font Replacement

<CodeBlock>

```css
/* Import your preferred fonts */
@import url('https://fonts.googleapis.com/css2?family=Poppins:wght@300;400;500;600;700&display=swap');
@import url('https://fonts.googleapis.com/css2?family=Fira+Code:wght@300;400;500;600;700&display=swap');

/* Override Kookie UI font variables */
:root {
  /* Replace base font definitions */
  --font-inter: 'Poppins', -apple-system, BlinkMacSystemFont, sans-serif;
  --font-jetbrains-mono: 'Fira Code', 'JetBrains Mono', monospace;
  
  /* Update theme font mappings */
  --font-sans: var(--font-inter);
  --font-mono: var(--font-jetbrains-mono);
}
```
</CodeBlock>

## Configuration

For dynamic font switching, use the provided utilities:

<CodeBlock>

```tsx
import { applyFontConfig, FONT_STACKS } from '@kushagradhawan/kookie-ui';

// Apply Google Fonts
applyFontConfig({
  sans: FONT_STACKS.poppins,
  mono: FONT_STACKS.firaCode,
});

// Use system fonts for performance
applyFontConfig({
  sans: FONT_STACKS.system,
  mono: FONT_STACKS.systemMono,
});
```
</CodeBlock>

<Text size="2" color="gray">
Available font stacks: <Code>inter</Code>, <Code>system</Code>, <Code>poppins</Code>, <Code>openSans</Code>, <Code>jetbrainsMono</Code>, <Code>firaCode</Code>, <Code>sourceCodePro</Code>, <Code>systemMono</Code>.
</Text>
