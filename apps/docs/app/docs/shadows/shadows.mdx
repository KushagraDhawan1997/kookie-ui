# Shadows

Custom shadow system with 6-step scale for elevation and hierarchy. Kookie UI Shadows are extracted from [Radix Themes](https://www.radix-ui.com/themes) with enhanced values for better depth perception. 

For complete shadow documentation and advanced usage, visit the [Radix Themes shadows docs](https://www.radix-ui.com/themes/docs/theme/shadows). Note that Kookie UI's shadow values may be enhanced and might not match exactly what's shown in the Radix documentation.

</Section>

<Section size="1">

## Usage

Use shadow tokens to create elevation and hierarchy. Each shadow level provides meaningful depth for different interface elements.

<CodeBlock>

```css
/* Card with subtle elevation */
.my-card {
  box-shadow: var(--shadow-2);
}
```

</CodeBlock>

<CodeBlock>

```css
/* Elevated button */
.my-button {
  box-shadow: var(--shadow-3);
}
```

</CodeBlock>

</Section>



<Section size="1">

## Tokens

Shadow tokens with their CSS values and usage:

<CodeBlock>

```css
/* Shadow tokens (see tokens/shadow.css for full values) */
var(--shadow-1)  /* subtle ring + inset; pressed states */
var(--shadow-2)  /* light elevation; classic cards */
var(--shadow-3)  /* medium elevation; raised cards, buttons */
var(--shadow-4)  /* small overlays, hover cards */
var(--shadow-5)  /* popovers, dropdowns */
var(--shadow-6)  /* large overlays, dialogs */
```

</CodeBlock>

</Section>
